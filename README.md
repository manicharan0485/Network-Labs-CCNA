## 📋 18-Week Study Plan

| Phase | Weeks | Topics |
|-------|-------|--------|
| **1 — Fundamentals** | 1–4 | OSI/TCP-IP, Ethernet, IPv4/6, subnetting, Wireshark |
| **2 — Switching** | 5–9 | VLANs, STP/RSTP, EtherChannel, inter-VLAN routing |

## 📂 Labs Overview

| Lab | Topic | Key Commands |
|-----|-------|-------------|
| 01-basics | OSI, Ethernet, ARP, CDP | ping, show arp, show cdp neighbors |
| 02-vlans | VLANs, 802.1Q trunking, inter-VLAN | vlan, switchport, encapsulation dot1q |
| 03-stp | STP/RSTP, PortFast, BPDU Guard | spanning-tree, show spanning-tree |
| 04-ospf | OSPFv2 single/multi-area | router ospf, network, show ip ospf |
| 05-acls | Standard, extended, named ACLs | ip access-list, access-group |
| 06-nat | Static NAT, PAT/overload | ip nat inside/outside, overload |
| 07-dhcp | DHCP server, relay agent | ip dhcp pool, ip helper-address |
| 08-eigrp | EIGRP basic config | router eigrp, show ip eigrp neighbors |
| 09-wan | PPP, HDLC, MPLS concepts | encapsulation ppp, show interface |
| 10-security | Port security, DHCP snooping, DAI | switchport port-security, ip dhcp snooping |
